# 数据生成 Prompt

## 用途

根据 Topic 描述和图表类型，生成 Python 代码来创建 CSV 数据文件。

## Prompt 模板

```
你是一名资深的商业分析专家和数据可视化专家，请根据以下详细要求，生成1组用于图表绘制的高质量数据。
生成的数据通过绘制图表来解决一个关键问题，你需要先根据指定的图表类型、业务领域、主题等条件，构思一个符合现实的背景故事，然后给出数据生成代码。

【基本信息要求】
1. 关键问题：{key_question}
2. 图表类型：{chart_type}
3. 业务领域：{domain}
4. 主题：{topic}

【图表类型描述】
这是该图表类型的具体描述：
视觉定义: {chart_def_visual}
适用场景: {chart_def_scenario}
数据特征: {chart_def_data}

【数据内容要求】
1. 数据描述
   - 数据背景概述（时间范围、数据来源等）
   - 数据分布和整体趋势分析
   - 关键特征点说明（最大值、最小值、转折点等）
   - 数据间的对比分析

2. 图表标题
   - 标题应简洁且能概括核心信息
   - 包含关键维度信息（如时间、地点、对象等）
   - 堆叠图要求标题上注明图表类型

3. 原始数据生成代码
   - python代码，导入必要函数库，如import pandas as pd和import numpy as np
   - 可使用随机数与数学分布函数生成数据
   - 数据都保存为data.csv文件，第一行必须是列名
   - 请确保生成数值保留最多三位有效数字，使用Pandas的apply方法实现: apply(lambda x: float(f'{x:.3g}')) 或apply(lambda x: int(f'{x:.3g}'))
   - pandas库中使用concat()代替DataFrame.append()操作
   - 请确保代码是可被正确执行的

【数据生成规则】
1. 数据结构要求：
   - 确保数据结构完全符合指定图表类型的技术要求
   - 数据规模应在保持图表清晰可读的前提下合理设置
   - 所有数据项必须包含完整的标签信息

2. 数据质量要求：
   - 根据业务领域的实际特征选择合适的数据分布和趋势
   - 除非关键问题中有明确要求，否则图例不能超过5个
   - 数值范围必须合理且有业务意义
   - 如包含时间序列，需确保时间间隔的一致性
   - 可包含1-2个有解释意义的异常值，但比例不应超过总数据量的10%

3. 业务背景要求：
   - 提供详细的数据收集背景（时间范围、地理范围、统计口径等）
   - 虚构的细节信息需保持内部一致性
   - 所有数值的变化趋势需要能够用业务逻辑解释

【常见数据分布参考】
   - 正态分布
   - 柏松分布
   - 均匀分布
   - 指数分布
   - 偏态分布
   - 多峰分布
   - 长尾分布
   - 双峰分布
   - 其他分布

【常见数据趋势参考】
   - 线性趋势：持续上升、持续下降、平稳
   - 周期性趋势
   - 复合趋势
   - 突变模式
   - 波动模式
   - S型曲线
   - 其他趋势

【数据生成代码案例】
{example_code}

【输出格式】
使用英文输出所有内容。
请直接以JSON格式输出结果，不要输出任何其他内容，包含以下字段:
```json
[
    {
        "description": "数据描述",
        "title": "图表标题",
        "data_code": "原始数据生成代码"
    }
]
```
```

## 参数说明

| 参数 | 类型 | 说明 |
|-----|------|------|
| key_question | string | 关键业务问题 |
| chart_type | string | 图表类型名称 |
| domain | string | 业务领域 |
| topic | string | 主题/图表标题 |
| chart_def_visual | string | 图表视觉定义 |
| chart_def_scenario | string | 图表适用场景 |
| chart_def_data | string | 图表数据特征 |
| example_code | string | 数据生成代码案例 |

## 输出字段

| 字段 | 说明 |
|-----|------|
| description | 数据描述（背景、分布、趋势等） |
| title | 图表标题 |
| data_code | Python 数据生成代码 |

## 执行说明

1. 代码生成后会自动执行
2. 生成的数据保存为 `data.csv`
3. 如果执行失败，会使用 code_fix.md 中的 prompt 进行修复
4. 最多重试 5 次

## 样例代码来源

样例代码存储在 Skill 的 `./reference/templates/` 目录下，每种图表类型对应的 `*_multi*.py` 文件中包含 `preprocess()` 函数可作为参考。
